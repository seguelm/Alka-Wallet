# Alka-Wallet
Alka Wallet - Módulo Front End

Flujo de uso de la aplicación
1. Login
Ingresar un email y una contraseña (validación básica)Si los campos están vacíos, se muestra un mensaje de error 
Al validar correctamente, se redirige al menú principal

2. Menú principal
Muestra el saldo generalPermite seleccionar entre:
Cuenta Corriente
Tarjeta de Crédito
Cuenta de Ahorro
El saldo y la fecha cambian según la cuenta seleccionada
Desde el menú se puede acceder a:
Enviar dinero
Añadir dinero
Movimientos

3. Enviar dinero
Ingresar destinatario (nombre o email)
Ingresar monto
Validación básica de campos
Muestra mensaje de éxito (simulación)

4. Añadir dinero
Ingresar monto a depositar
Validación básica
Muestra mensaje de confirmación (simulación)


5. MovimientosVisualización de movimientos recientes
Ingresos y egresos diferenciados visualmente
Montos alineados y destacados

Notas importantes:La aplicación es una simulación frontend, no realiza transacciones reales
